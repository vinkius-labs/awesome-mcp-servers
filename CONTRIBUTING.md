# Contributing

To add a new MCP server to this list:

1. Fork the repository
2. Add the server to the appropriate category in `README.md`
3. Use the format: `| [name](https://vinkius.com/en/apps/name) | Description |`
4. Keep descriptions under 50 characters
5. Open a Pull Request

## Adding a Category

If your server does not fit any existing category, propose a new one. Categories must have at least 5 servers.

## Guidelines

- Servers must be available on the [Vinkius AI Gateway](https://vinkius.com)
- Descriptions should be factual and concise
- No promotional language
